# gdansk-oswiadczenia
Scraper Oświadczeń Majątkowych - Gdańsk 2026 Program pozwala na automatyczne scrapowanie oświadczeń majątkowych radnych Rady Miasta w Gdańsku udostępnionych w 2026 roku.

## Jak użyć?

### Opcja 1: Uruchomienie w przeglądarce (Google Colab)
Nie musisz nic instalować na swoim komputerze:
1. Kliknij plik **[Gdansk_oswiadczenia.ipynb](Gdansk_oswiadczenia.ipynb)** w tym repozytorium.
2. Kliknij przycisk **"Open in Colab"**, który pojawi się nad kodem.
3. W Colabie klikaj przycisk "Play" przy kolejnych komórkach.
4. Po zakończeniu, pobrane pliki znajdziesz w ikonie folderu (po lewej stronie w Colabie).

### Opcja 2: Uruchomienie lokalne
Jeśli wolisz uruchomić skrypt u siebie na komputerze:
1. Pobierz plik `Gdansk_oswiadczenia.ipynb` (możesz go przekonwertować na `.py`) lub korzystaj bezpośrednio ze swojego pliku `gdansk2.py`.
2. Zainstaluj wymaganą bibliotekę:
   
```bash
   pip install requests
Następnie uruchom skrypt:
